## 3. 判定パターン整理

判定に使われる条件式は以下の4パターンに集約される。

| パターン | 条件式 | 主な該当入院料 |
|---|---|---|
| **P1** | A≥3 または C≥1 | 急性期一般1（基準①）、特定機能7対1・専門7対1（基準①） |
| **P2** | (A≥2 かつ B≥3) または A≥3 または C≥1 | 急性期一般2〜5、10対1系、脳卒中CU、地域包括医療 |
| **P3** | A≥1 または C≥1 | 特定一般※7、地域包括ケア |
| **P4** | A≥2（ICU/HCU系独自） | 特定集中治療室、救命救急2・4 |
| **P5** | HCU独自項目のいずれかに該当 | HCU、救命救急1・3 |
| **P6** | B項目（診療・療養上の指示 または 危険行動） | 看護職員夜間配置加算（地域包括ケア） |
| **P7** | 特殊な治療法等に該当 | 重症患者対応体制強化加算 |
