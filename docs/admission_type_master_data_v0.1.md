## 1. judgment_pattern_master（判定パターンマスタ）

今回使用するのは P1〜P3 の3パターン。

| id | code | label |
|----|------|-------|
| 1 | P1 | A≧3 または C≧1 |
| 2 | P2 | (A≧2 かつ B≧3) または A≧3 または C≧1 |
| 3 | P3 | A≧1 または C≧1 |

> P4〜P7 は将来の ICU/HCU/加算対応時に追加する。
